Inciso 1 Importacin, limpieza de datos y dispersin

Se import el archivo A1.6 Tiempo de Entrega.csv y se aplic una limpieza de atpicos usando el criterio IQR tanto en la variable Calificacion como en Tiempo.
El dataset pas de 432 registros a 376 registros despus de la limpieza.

La grfica de dispersin muestra una clara tendencia: conforme el tiempo de entrega se acerca al cierre de la bandeja, la calificacin tiende a ser ms baja, aunque con cierta dispersin en los valores.

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("A1.6 Tiempo de Entrega.csv")

# Limpieza de atpicos (IQR) en Calificacion
Q1 = df["Calificacion"].quantile(0.25)
Q3 = df["Calificacion"].quantile(0.75)
IQR = Q3 - Q1
df = df[(df["Calificacion"] >= Q1 - 1.5*IQR) & (df["Calificacion"] <= Q3 + 1.5*IQR)]

# Limpieza de atpicos (IQR) en Tiempo
Q1_t = df["Tiempo"].quantile(0.25)
Q3_t = df["Tiempo"].quantile(0.75)
IQR_t = Q3_t - Q1_t
df = df[(df["Tiempo"] >= Q1_t - 1.5*IQR_t) & (df["Tiempo"] <= Q3_t + 1.5*IQR_t)]

print("Tamao despues de limpiar:", df.shape)

plt.scatter(df["Tiempo"], df["Calificacion"], s=12)
plt.xlabel("Tiempo")
plt.ylabel("Calificacion")
plt.title("Dispersion Tiempo vs Calificacion (sin atipicos)")
plt.grid(True)
plt.show()

from sklearn.pipeline import Pipeline
from sklearn.neighbors import KNeighborsRegressor
from sklearn.preprocessing import StandardScaler
import numpy as np

best_k, best_rmse = None, float("inf")
for k in [3,5,7,9,11]:
    knn = Pipeline([("scaler", StandardScaler()), ("knn", KNeighborsRegressor(n_neighbors=k))])
    knn.fit(X_tr, y_tr)
    pred = knn.predict(X_tr)
    rmse = float(np.sqrt(np.mean((y_tr - pred)**2)))
    if rmse < best_rmse:
        best_rmse, best_k = rmse, k

knn_best = Pipeline([("scaler", StandardScaler()), ("knn", KNeighborsRegressor(n_neighbors=best_k))]).fit(X_tr, y_tr)

Conclusin General

La actividad permiti analizar la relacin entre el tiempo de entrega y la calificacin aplicando distintos modelos de regresin no lineal. Despus de limpiar los datos y dividirlos en entrenamiento y prueba, se comprob que los subconjuntos eran consistentes. Entre los modelos evaluados (polinomial, segmentado y KNN), el polinomial de grado 3 mostr el mejor desempeo con el menor RMSE y una curva ms estable e interpretativa, mientras que el segmentado ofreci un resultado muy cercano pero sin superar al polinomial, y el KNN present mayor variabilidad y menor interpretabilidad. En conclusin, aunque el modelo polinomial es el que ms se ajusta a los datos, esto no comprueba de manera definitiva la hiptesis sobre la relacin entre entregar ms tarde y obtener menor calificacin, sino que nicamente sugiere una posible tendencia que debera validarse con ms evidencia.
